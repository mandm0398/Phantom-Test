Rev Robotics Wall-E Robot

During the summers of 2016 and 2017, I worked at Rev Robotics, an educational robotics company which provides parts and control systems for high school robotics teams participating in the FIRST Tech Challenge. While there, I worked on various projects ranging from improving a robotics kit meant to be used to teach robotics in the classroom to developing example robots using only Rev parts. One of these example robots was a working Wall-E robot which was meant for promotional purposes.
